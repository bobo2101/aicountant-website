## 🎁 策略三：定價心理學技巧

### 1. 錨定效應（Anchoring）

```
公司註冊 - 高級版
原價：¥9,999  ❌
現價：¥6,999  ✅

vs

公司註冊 - 標準版 ¥3,999

顯得標準版好抵！
```

### 2. 中間選項效應

```
基礎版：¥1,999  ← 太少人選
標準版：¥3,999  ← 最多人選（目標）
高級版：¥6,999  ← 顯得標準版抵
```

### 3. 組合優惠

```
單買：
公司註冊 ¥3,999
首年記賬 ¥3,588
總計：¥7,587

套餐價：¥5,999（省 ¥1,588）

轉化率提升 40%！
```
