## 🚀 立即開始

### 第一步：你去 Stripe 註冊（約 20 分鐘）

1. 去 https://dashboard.stripe.com/register
2. 選擇 **Hong Kong** 作為地區
3. 填寫公司信息
4. 驗證郵箱
5. 去 Developers → API keys 複製：
   - `pk_test_...` (Publishable key)
   - `sk_test_...` (Secret key)
6. 去 Settings → Payment methods 啟用：
   - ☑️ Cards
   - ☑️ Alipay
   - ☑️ WeChat Pay

**完成後將兩個 keys 發俾我！**
